# WiBaSets
Basque data for NLP and ML

Text Data
-----------
Master file (master.txt) includes 5,129,348 Basque tokens extracted from news articles, wikipedia pages, book reviews, and more. 

CSV data
-----------
There are several files for sentiment analysis models. We add a trainning, test, and validation files for such purposes. All utterances are labeled as positive (1), negative (-1), or neutral (0).
